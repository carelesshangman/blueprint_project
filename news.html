Blueprint is a smart assistant developed by Jaka German. The idea behind the machine was inspired by multiple robot-like characters in movies and videogames, for example HAL 9000 from 2001: A Space Odyssey or GLaDOS from Valve's Portal series. Main difference between these fictional characters and Blueprint is the evil nature of the characters. While GLaDOS and HAL had no intention in helping their hosts, Blueprint provides useful functions, like music, internet browsing and even helps out with programming and electronics if you ask it to.

"When you understand how to make a robot companion, the code writes itself" - Jaka German
By observing similar projects I was able to learn so much. My main takeaway from Project Blueprint was that learning new programming skills really isn't difficult. In a matter of months I went from an average JavaScript user to a quite adept one.
